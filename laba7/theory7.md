## 1. Общие сведения о STL

STL (Standard Template Library) — стандартная библиотека шаблонов C++,  
предназначенная для удобной и эффективной работы с данными.

STL состоит из трех основных компонентов:
- **Контейнеры** — структуры данных для хранения элементов;
- **Итераторы** — универсальный механизм доступа к элементам контейнеров;
- **Алгоритмы** — обобщённые функции обработки данных.

---

## 2. Контейнеры STL

Контейнер — объект, содержащий набор однотипных элементов.

Контейнеры STL бывают:
- последовательные (`vector`, `list`, `deque`);
- ассоциативные (`set`, `map` и др.).

В данной лабораторной работе используется **последовательный контейнер `vector`**.

### Контейнер vector
`vector` — динамический массив:
- автоматически управляет памятью;
- поддерживает доступ по индексу `[]` и `at()`;
- эффективно работает с итераторами;
- поддерживает добавление элементов через `push_back()`.

Пример объявления:
```cpp
#include <vector>
vector<int> v;
```

## 3. Итераторы

Итераторы — это обобщённые указатели, предназначенные для доступа к элементам контейнеров.

Основные операции с итераторами:
- *it — доступ к элементу;
- ++it — переход к следующему элементу;
- it1 == it2, it1 != it2 — сравнение;
- присваивание итераторов.

Пример:
```cpp
vector<int>::iterator it;
for (it = v.begin(); it != v.end(); ++it)
    cout << *it;
```

Методы контейнера:

`begin()` — итератор на первый элемент;

`end()` — итератор на элемент за последним.

## 4. Алгоритмы STL

Алгоритмы STL — это шаблонные функции, работающие с диапазонами элементов.

Для использования необходимо подключить:

`#include <algorithm>`


Диапазон задаётся полуинтервалом:

`[first, last)`

## 5. Обязательные алгоритмы в работе
### 5.1 for_each

Выполняет действие над каждым элементом диапазона.

`for_each(v.begin(), v.end(), func);`

### 5.2 find_if

Находит первый элемент, удовлетворяющий условию (предикату).

`it = find_if(v.begin(), v.end(), predicate);`

### 5.3 transform

Преобразует элементы диапазона и сохраняет результат.

`transform(v.begin(), v.end(), v.begin(), func);`

## 6. Предикаты и функциональные объекты

Предикат — функция или объект, возвращающий bool.

Пример:
```cpp
bool isPositive(int x) {
    return x > 0;
}
```

Используется в алгоритмах `find_if`, `remove_if` и др.
